# 2. Pitch Deck & Script

## Pitch Goal

In 5 minutes, prove three claims:

1. Journalists face publication pressure before evidence is settled.
2. Lookitup gives them a trusted-source evidence workspace.
3. The prototype proves the core search loop.

## Main Pitch Message

**Lookitup stops unverified claims before they become published news.**

## Catchy Tagline

**Turn doubt into evidence before publishing.**

### Title

### Figure

## Slide 4 - Solution

### Title

**Lookitup: trusted-source evidence before publication.**

### Figure

Use `assets/figures/figure-04-core-loop.png`.

### Content

Core loop:

1. Select a Trusted Source Pack.
2. Search a claim or subject.
3. Read Evidence Cards.
4. Decide before publishing.

### Script

A newsroom, desk editor, fact-checking team, or independent journalist defines trusted sources before the deadline.

During breaking news, the journalist selects a source pack, searches the claim, and sees Evidence Cards from that boundary.

## Backup Q&A

| Question | Answer |
| --- | --- |
| Why not Google? | Google searches the whole web. Lookitup searches only inside sources the journalist chose before the deadline. |
| Why not ChatGPT? | ChatGPT gives generated text. Lookitup shows source-bound evidence. |
| Is this automatic fact-checking? | It is pre-publication evidence search. The journalist decides. |
| Why use a local corpus in the demo? | The pitch must not depend on network access. URL/RSS ingestion is a later extension. |
| What is the technical core? | Controlled retrieval: source packs define the search boundary, and Evidence Cards make evidence visible. |
